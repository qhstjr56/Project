Introduction¶

The primary objective of this project is to develop a predictive model for the target variable, "Weather Type", based on a given set of features in the dataset. Accurate predictions of weather types can provide valuable insights for applications such as climate analysis, forecasting, and decision-making in weather-dependent industries.

To achieve this goal, we will undertake the following steps:

Exploratory Data Analysis (EDA): Understand the dataset's structure, assess data quality, and identify patterns or relationships among features.

Feature Engineering: Preprocess the data to handle missing values, categorical variables, and outliers. Engineer new features to enhance predictive power.

Model Development: Train multiple machine learning models to identify the one that performs best for classifying "Weather Type."

Model Evaluation: Evaluate model performance using metrics such as accuracy, F1-score, and precision-recall analysis. Use cross-validation to ensure generalization across datasets.

Threshold Optimization: Fine-tune model thresholds to improve class-specific predictions and maximize relevant metrics.

Model Optimization and Deployment: Implement strategies to ensure the final model is robust, interpretable, and ready for application.

By the end of this project, we aim to present a well-validated predictive model for "Weather Type" that achieves optimal performance while providing actionable insights into the relationships between features and weather classifications.

for i in range(n_classes):
    # Compute precision, recall, and thresholds for each class

    best_f1 = 0
    best_threshold = 0

    # Iterate over thresholds to compute F1-score
    for threshold in thresholds:
        # Generate binary predictions for the current threshold
        y_pred_class_i = (y_probs[:, i] >= threshold).astype(int)
        y_true_class_i = (y_test == i).astype(int)

        # Compute F1-score using the library
        score = f1_score(y_true_class_i, y_pred_class_i)

        # Update the best F1-score and threshold
        if score > best_f1:

            best_threshold = threshold

    # Store the best threshold for the current class
    best_thresholds.append(best_threshold)
